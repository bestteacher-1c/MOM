Специальный коэффициент для амортизации ОС (налог. учет)

В регистре хранятся значения специального коэффициента для начисления амортизации по налоговому учету согласно пп. 7, 8 и 9 статьи 259 НК РФ.
Данные регистра влияют на начисление амортизации по объектам ОС в налоговом учете - значение специального коэффициента определяется на начало месяца начисления амортизации.

Запись в регистр производится документами Ввод начальных остатков, Принятие к учету - при вводе первоначальных сведений по объекту, документом Изменение параметров - при изменении коэффициента для использования в следующем налоговом периоде.
