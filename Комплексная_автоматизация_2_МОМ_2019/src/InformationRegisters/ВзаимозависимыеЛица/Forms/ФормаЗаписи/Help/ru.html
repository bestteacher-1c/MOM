Взаимозависимое лицо

Хранит информацию о взаимозависимых лицах организаций.

Данная информация необходима для составления уведомления о контролируемых сделках: в соответствии с п.1 ст.105.14 НК РФ, контролю могут подлежать сделки с взаимозависимыми лицами, а также сделки с независимым посредником.

Для контрагентов, являющихся взаимозависимыми, указывается тип взаимозависимости и дата, начиная с которой он действует. Возможны следующие типы взаимозависимости:

  • Взаимозависимы по Кодексу (п.2 ст.105.1 НК РФ);
  • Самостоятельное признание взаимозависимости (п.6 ст. 105.1 НК РФ);
  • Взаимозависимы по решению суда (п.7 ст. 105.1 НК РФ);
  • Не взаимозависимый посредник (п.1.1 ст.105.14 НК РФ);

Для прекращения действия отношений зависимости нужно добавить еще одну запись, в которой выбрать тип Не взаимозависимы и дату, начиная с которой контрагент перестает быть взаимозависимым с организацией.

Управляет взаимозависимостью двух лиц. Требует указания организации (для которой устанвливается взаимозависимое лицо) и самого взаимозависимого лица. Необходим для предоставления уведомления о контролируемых сделках.

В качестве взаимозависимого лица может выступать как контрагент, так и организация. В начале указываем для какого типа данных указываем взаимозависимость (контрагент или организация), а потом сам объект, который является взаимозависимым.

Если для ранее указанного объекта меняются условия взаимозависимости (контрагент перестал быть взаимозависимым или из взаимозависимого лица стал независимым посредником), тогда создается новая запись регистра с датой начала действия соответствующей началу действия новых условий взаимозависимости.

Помимо данных о взаимозависимости, так же представлены данные об учетной политике контрагента. При этом, если контрагент является юридическим лицом, будут отображены именно данные об его учетной политики, если же он является физическим лицом, будут доступны для редактирования только вспомогательные данные для дополнительной информации по физическому лицу.

В случае, когда данные по взаимозависимому лицу открываются для уже ранее указанного лица (из списка регистра сведений или из отчета), взаимозависимое лицо не доступно для редактирования и можно корректировать лишь данные о типе взаимозависимости, учетной политике, а так же начала действия данных.
