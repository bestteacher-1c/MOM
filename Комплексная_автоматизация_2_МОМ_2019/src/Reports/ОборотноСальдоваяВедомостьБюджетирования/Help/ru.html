Оборотно-сальдовая ведомость по показателям бюджетов

Отчет предназначен для план-фактного анализа показателей бюджетов и их аналитик с учетом связей со статьями бюджетирования. Конечное сальдо по показателям рассчитывается как начальное сальдо плюс обороты по статьям.

Особенности отчета:

  • Отчет формируется по данным плановых/фактических сценариев бюджетирования.
  • Расшифровка сумм (например, до регистратора).

Настройки отчета позволяют в качестве группировки данных вывести дату актуальности плановых данных.
