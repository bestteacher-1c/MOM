Форма изменения данных по предмету сделки

Форма настройки данных по предмету сделки представляет собой простую форму, в которой представлены только те реквизиты элемента справочника номенклатуры, которые необходимы для составления уведомления о контролируемых сделках.

Классификаторы номенклатуры

Классификаторы номенклатуры служат для справочной информации по номенклатуре при предоставлении данных о контролируемых сделках в контролирующие органы. Они могут назнаться как непосредственно - в форме элемента справочника "Номенклатура", так и в этой форме. При этом ввод кодов по ОКП и ТНВЭД будет доступно только тогда, когда номенклатура является товаром.

Обложение по НДПИ по процентной ставке

Данный реквизит не отображается в элементе справочника "Номенклатура", однако необходим, для корректного предоставления уведомления о контролируемых сделках. Его редактирование доступно только из подсистемы "Контролируемые сделки" и его использование предусмотрено в ней же.

Это свойство говорит нам, что предмет сделки является полезным ископаемым и сделка по этому предмету будет облагаться налогом на добычу полезных ископаемых. При условии, что организация или контрагент тоже будут являться плательщиком налога на добычу полезных ископаемых, сделка будет относиться к контролируемой на основании суммы, представленной в регистре сведений "Границы контролируемости сделок", как сумма с особенностью отнесения - "Взаимозависимые лица на спецрежимах" (Одно из взаимозависимых лиц является плательщиком ЕНВД или ЕСХН).

Редактирование этого флажка доступно лишь когда предмет сделки является товаром.
