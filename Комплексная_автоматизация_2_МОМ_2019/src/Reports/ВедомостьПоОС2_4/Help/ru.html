Ведомость по ОС

Отчет предназначен для анализа стоимости и амортизации основных средств предприятия и содержит информацию о стоимости и амортизации на начало и конец периода, уменьшение или увеличение стоимости и амортизации за период.

Отчет формируется, начиная с даты начала учета версии 2.4. Для получения данных до этой даты можно использовать другой отчет Ведомость по ОС версии 2.2.

На увеличение или уменьшение стоимости и амортизации влияют события, которые действительно их меняют. Например, переоценка, начисление амортизации, списание. Перемещение не влияет на эти поля.

Варианты формирования

Для удобства просмотра предусмотрено несколько вариантов отчета:

  • Ведомость по ОС (по всем учетам)
  • Ведомость по ОС (бухгалтерский и налоговый учет)
  • Ведомость по ОС (бухгалтерский учет)
  • Ведомость по ОС (налоговый учет)
  • Ведомость по ОС (управленческий учет)
