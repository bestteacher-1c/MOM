Анализ расходов по УСН

Отчет предназначен для наглядного представления признания расходов организаций на упрощенной системе налогообложения с объектом налогообложения Доходы минус расходы.
Есть несколько вариантов настроек, разделенных по принципу механизма признания:

  • Расходы по товарам - можно анализировать данные по поступлению расходов, оплате и их отгрузке. Расходы признаются только тогда, когда выполнены все условия признания - все три суммы заполнены. При этом выводятся и суммы оставшиеся к признанию с прошлых периодов.
  • Расходы по материалам - анализируются данные только по поступлению расходов и их оплате. Расходы признаются при наличии поступления расхода и его оплаты. При этом выводятся и суммы оставшиеся к признанию с прошлых периодов.
  • Прочие расходы - Аналогично предыдущему варианту, расходы признаются при наличии поступления расхода и его оплаты, однако только для тех случаев, когда в статье расходов непосредственно указана необходимость признания данного вида расходов для целей УСН. При этом выводятся и суммы оставшиеся к признанию с прошлых периодов.
  • Расходы по ОС и НМА - в отличие от предыдущих случаев, содержит дополнительную информацию по элементу расхода (данные справочников ОС и НМА), факту принятия к учету (для ОС). Помимо этого отличается и механизм признания расходов: они признаются равными долями в течении года с момента их оплаты.

Расшифровки отчета позволяют переходить к исходным документам по поступлению, оплате и отгрузке.
