Сверка данных оперативного и бухгалтерского учета

Отчет предназначен для выявления и представления в табличном формате отклонений остатков и/или оборотов оперативного учета от регламентированного бухгалтерского учета за отчетный период.

Примечание. Отчет может показывать расхождения, вызванные разной детализацией учетов, когда в оперативном учете обороты представлены развернуто, а в регламентированном учете - свернуто. Например, в оперативном учете до объекта расчета и расчетного документа, а в регламентированном до субконто Договор. При этом такие отклонения по оборотам в итоге дают ноль и, в общем случае, ошибкой не являются.

Отчет может быть использован как инструмент аудита для решения следующих задач:

  • Просмотр оборотов оперативного учета и проведение сравнительного анализа отклонений в суммах оборотов оперативного и регламентированного бухгалтерского учета с разбивкой по Разделам учета, обосновывающих разницу в регламентированном и оперативном учете.
  • Расшифровка показателей с детализацией до документа, которая показывает какой документ вызвал расхождения.

Разделы учета

Отчет структурирован по следующим разделам учета:

Денежные средства

Себестоимость товаров

Расчеты с поставщиками (Расчеты с клиентами)

Расчеты по кредитам и депозитам

Прочие расчеты с контрагентами

Расходы

Денежные средства


Источники данных

  • Регистр накопления «Денежные средства (безналичные)»
  • Регистр накопления «Денежные средства (наличные)»
  • Регистр накопления «Денежные средства в кассах ККМ»
  • Регистр накопления «Денежные средства у подотчетных лиц»
  • Регистр накопления «Денежные средства в пути»
  • Регистр накопления «Расчеты по договорам кредитов и депозитов» (в части договоров депозитов)
  • Регистр бухгалтерии «Хозрасчетный» (счета 50, 51, 52, 55, 57, 71)


Особенности

  • Сверка производится только в валюте регламентированного учета
  • Субконто по счету 50 нет, поэтому сверка проводится по всем кассам предприятия

Себестоимость товаров


Источники данных

  • Регистр накопления «Себестоимость товаров» (измерение «Раздел учета» не равно «Производственные затраты»,«Незавершенное производство», «Товары принятые на комиссию», «Материалы принятые в переработку»)
  • Регистр бухгалтерии «Хозрасчетный» (все балансовые счета, у которых субконто «Номенклатура» и «Склад», а также 10.07, 15, 45)


Особенности

  • Методика отражения движений в регистрах оперативного учета и в регламентированном отличается, поэтому для документов, отражающих хозяйственные операции «ВозвратИзПроизводства», «ВозвратМатериаловИзКладовой», «ВозвратТоваровОтКлиента», «ВозвратОтКомиссионера», «ВозвратОтПереработчика», «ВозвратДавальцу», «ВозвратТоваровКомитенту» для оперативного учета «Приход» и «Расход» поменяны местами с изменением знака.
  • Субконто «Склад» на субсчетах учета номенклатуры оборотное. Для корректной работы отчета нужно исправить на необоротное.

Расчеты с поставщиками (Расчеты с клиентами)


Источники данных

  • Регистр накопления «Расчеты с поставщиками по документам» («Расчеты с клиентами по документам»)
  • Регистр накопления «Себестоимость товаров» (Раздел учета «Неотфактурованные поставки»)
  • Регистр бухгалтерии «Хозрасчетный» (счет 60 (62) с субсчетами)


Особенности

  • Сравнение ведется только в валюте регламентированного учета
  • Методика отражения возвратов в оперативном и регламентированном учетах разная – в регламентированном учете «сторно», в оперативном «расход». Поэтому обороты по оперативному учету откорректированы таким образом, что бы они совпадали с бухгалтерским.
  • В оперативном учете отсутствует необходимость в движениях, закрывающих авансы. Соответственно в обороты оперативного учета при необходимости добавляется «виртуальный» оборот, имитирующий операцию закрытия аванса в регламентированном учете.

Расчеты по кредитам и депозитам


Источники данных

  • Регистр накопления «Расчеты по договорам кредитов и депозитов» (кроме договоров депозитов)
  • Регистр бухгалтерии «Хозрасчетный» (счета: 58.03, 66, 67, 76.03)


Особенности

  • Сверка производится только в валюте регламентированного учета

Прочие расчеты с контрагентами


Источники данных

  • Регистр накопления «Расчеты с поставщиками по документам»
  • Регистр накопления «Расчеты с клиентами по документам»
  • Регистр бухгалтерии «Хозрасчетный» (все возможные счета расчетов с контрагентами кроме счетов 60 и 62)


Особенности

  • Сравнение ведется только в валюте регламентированного учета
  • В том случае, когда операция взаимозачета задолженности отражается через счет 76.09, в регистрах оперативного учета отсутствует соответствующий оборот. В этом случае отчет всегда будет показывать разницу, что нормально.

Расходы


Источники данных

  • Регистр накопления «Себестоимость товаров» (когда измерение «Раздел учета» = «Производственные затраты» или «Незавершенное производство»)
  • Регистр накопления «Прочие расходы»
  • Регистр накопления «Партии прочих расходов» (по статьям расходов с вариантом распределения "На себестоимость товаров")
  • Регистр накопления «Прочие расходы незавершенного производства»
  • Регистр накопления «Трудозатраты незавершенного производства»
  • Регистр бухгалтерии «Хозрасчетный» (счета 20, 23, 25, 26, 28, 29, 44, 91.02, 94, 97, 10.ДР, 41.ДР)


Особенности

  • Аналитика на счетах затрат – оборотная (кроме вида субконто «Группы (виды) продукции»), поэтому сверка производится без аналитики (но с расшифровкой до регистратора)
  • Статьи расходов с неиспользуемым более вариантом распределения На прочие активы и аналитикой Прочие активы не анализируются по оперативному учету, как соответствующие статьям активов. Статьи расходов с вариантом распределения На внеоборотные активы не имеют отражения по регламентированным суммам в оперативном учете и с бухгалтерским учетом так же не сравниваются.
