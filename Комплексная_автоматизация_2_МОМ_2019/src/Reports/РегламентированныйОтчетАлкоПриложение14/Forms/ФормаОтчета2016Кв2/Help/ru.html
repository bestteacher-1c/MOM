Отчет «Декларация об объеме собранного винограда для производства винодельческой продукции»

Декларация об объеме собранного винограда для производства винодельческой продукции заполняется сельскохозяйственными товаропроизводителями, организациями, не признаваемыми сельскохозяйственными товаропроизводителями, гражданами, ведущими личное подсобное хозяйство, осуществляющими выращивание винограда для производства винодельческой продукции.

Особенности заполнения:

Титульный лист.

Необходимо проверить сведения об организации и сведения о должностных лицах, которые были заполнены автоматически и, при необходимости, отредактировать данные в справочнике Организации.

 

Декларация об объеме собранного винограда для производства винодельческой продукции

В Разделе Декларация необходимо ввести сведения об объеме собранного винограда и количествах отгруженного винограда получателям.

Данные по каждому обособленному подразделению или винограднику, если отчет заполняется не юридическим лицом, вносятся в строку "Сведения по обособленному подразделению организации, не признаваемой сельскохозяйственным товаропроизводителем, сельскохозяйственного товаропроизводителя, гражданина, ведущего личное подсобное хозяйство (место нахождения виноградника)".

Для добавления новой строки необходимо использовать кнопки "+" или "Добавить строку", расположенную непосредственно под многострочной частью отчета, либо воспользоваться контекстным меню, выделив любую ячейку добавляемой строки.

Для удаления строк необходимо использовать кнопки "х", расположенные слева от строк многострочных частей отчета, либо воспользоваться контекстным меню, выделив любую ячейку удаляемой строки.

Если организация не имеет обособленных подразделений (или один виноградник), то необходимо заполнить сведения по организации в нижней таблице Декларации.

В случае, если обособленных подразделений (или виноградников) несколько, необходимо добавить дополнительные страницы с доступной для ввода данных нижней частью таблицы.

В верхней таблице Декларации "Сведения по организации, не признаваемой сельскохозяйственным товаропроизводителем, сельскохозяйственному товаропроизводителю, гражданину, ведущему личное подсобное хозяйство" указывается сводная информация по всем обособленным подразделениям (виноградникам). Заполняется только на первой странице. Если орагнизация не имеет обособленных подразделений (или один виноградник), то в верхней таблице дублируется информация из заполненной нижней.


Для управления дополнительными страницами (добавление/удаление) необходимо использовать кнопки, расположенные вверху страницы, либо воспользоваться контекстным меню в панели навигации по разделам отчета, выделив нужную страницу.
