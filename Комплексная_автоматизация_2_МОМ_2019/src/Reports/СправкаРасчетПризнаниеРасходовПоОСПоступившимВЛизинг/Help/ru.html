Справка-расчет "Признание расходов по ОС, поступившим в лизинг"

Отчет предназначен для обобщения информации о расходах, связанных с основными средствами, поступившими по договорам лизинга.

Отчет рекомендуется формировать только после выполнения регламентных операций закрытия месяца.


В отчет выводятся данные:

  • Контрагент, Договор,
  • Задолженность по аренде,
  • Сумма амортизации,
  • Сумма признанных расходов по аренде,
  • Сумма признанных расходов по арендным платежам в налоговом учете,
  • Корректировка суммы амортизации в налоговом учете.


Параметры формирования отчета можно задать по кнопке Показать настройки.
