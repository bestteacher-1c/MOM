Справочник "Сайты"

 Содержит данные сайтов, созданных из приложения 1С. Также справочник используется для настройки переходов на сайт из 1С: в интерфейсе раздел "Сайт компании - Перейти на сайт".

Для настройки переходов, следует указать адрес сайта и адрес админзоны (раздела для администратора сайта) в соответствующих полях справочника. После этого настройка перехода на сайт появится в списке сайтов  (по нажатию "Перейти на сайт"), можно открыть сайт сразу из приложения 1С.

Также на основании справочника Сайты можно создать Узел обмена с сайтом, в котором будут заполнены имеющиеся данные.
