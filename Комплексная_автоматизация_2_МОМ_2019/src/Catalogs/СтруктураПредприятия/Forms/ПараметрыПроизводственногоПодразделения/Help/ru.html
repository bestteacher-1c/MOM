Основные параметры

Признак Производит продукцию устанавливается для подразделений, участвующих в производственном процессе. Если процесс подразумевает материальный выход, то от имени подразделения оформляются документы выпуска продукции. Материалы в подразделение передаются документами Передачи материалов в производство, а их распределение на выпуск - документами Списание затрат на выпуск или Распределение материалов и работ.

Дополнительно, для производственного подразделения указывается:

  • График работы - определяет график работы, по которому работает подразделение.
  • Склад материалов - определяет склад, с которого передаются материалы в производство.

Производство без заказов (производство версии 2.1)

Настройка доступна для управления производством версии 2.1.
При выпуске продукции без заказов на производство предусмотрено два способа отнесения затрат на выпуск:

  • Отдельным документом Списание затрат на выпуск, в котором указывается продукция, перечень материалов и выполненных работ сотрудников, которые требуется отнести на указанную продукцию. Документ вводится вручную или регламентным заданием. Предусмотрены различные сервисы заполнения документа по нормативам и обособлению. Такой способ отнесения затрат рекомендуется для производств, в которых требуется управлять себестоимостью партий выходных изделий.
  • По правилам распределения, которые задаются для материалов и работ в рабочем месте Распределение материалов и работ. Для отнесения затрат на выпуск без заказов могут быть задействованы следующие базы распределения: Количество продукции, Объем продукции, Вес продукции, Плановая стоимость продукции. Непосредственное отнесение затрат на выпуск, то есть списание из свободного остатка в подразделении, выполняется при закрытии месяца. Регистрация трудозатрат в таких подразделениях не предусмотрена. Этот способ отнесения расходов позиционируется для простых, вспомогательных производств.

Учет работ сотрудников

На закладке Учет работ сотрудников предумотрен ряд настроек для регистрации и учета производственных работ.

Для выполнения производственных операций в подразделении могут привлекаться работники предприятия персонально или коллективы исполнителей - бригады. В зависимости от выбранной настройки в документах производства будет доступен тот или иной тип исполнителя.

Для бригад доступны настройки распределения суммы по членам бригады - работникам. Распределение может выполняться по коэффициентам трудового участия, отработанному времени и тарифным ставкам работников.

Выработка бригад и работников может оформляться за произвольный период, раз в месяц или раз в день. В зависимости от выбранной настройки создаются документы выработки сотрудников при групповой обработке строк или при оформлении документов из формы закрытия месяца.
