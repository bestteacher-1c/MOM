Убытки прошлых лет

Справочник предназначен для ведения аналитического учета убытков прошлых лет по годам возникновения.

Справочник входит в вид субконто "Расходы будущих периодов" и используется для ведения аналитического учета на счете 97.11 "Убытки прошлых лет".

На каждый год убытка создается отдельный элемент справочника. При переносе убытка на будущее в регламентной операции закрытия года элемент справочника создается автоматически.
