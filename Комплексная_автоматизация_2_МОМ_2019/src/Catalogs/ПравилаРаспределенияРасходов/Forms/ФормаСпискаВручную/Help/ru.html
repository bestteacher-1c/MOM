Форма списка вручную

В форме выводятся показатели для распределения расходов и их значения.
При изменении рабочего периода выводятся показатели действующие на данный период.

Кнопка Показывать только незаполненные показатели позволяет вывести показатели для которых не указаны значения:

  • для показателей вводимых ежемесячно - должны быть заданы значения в выбранном рабочем периоде;
  • для показателей вводимых при изменении - должно быть задано хотя бы одно значение показателя.
