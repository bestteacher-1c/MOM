Правило распределения расходов

Справочник предназначен для описания правил распределения постатейных расходов, материалов и работ на себестоимость продукции. Настройка распределения может применяться для:

  • распределения постатейных расходов между подразделениями, на партии производства внутри подразделений и на другие статьи расходов
  • распределения материалов и работ на партии производства
  • описания показателей распределения постатейных расходов между подразделениями.

Форма предназначена для описания настроек распределения материалов и работ. Настройка применяется в документе Распределение материалов и работ на себестоимость продукции. Для распределения могут использоваться следующие данные:

Количество (объем, вес) указанных материалов

Количество (объем, вес) продукции

Плановая стоимость продукции

Количество работ указанных видов

Норматив расхода данного материала

Количество (объем, вес) указанных материалов

В качестве базы распределения используются данные о расходе указанных материалов в производстве в течение месяца. Для базы По объему, весу количество материала умножается на коэффициент объема (веса) из карточки номенклатуры. Список материалов задается по соответствующей гиперссылке.

Количество (объем, вес) продукции

Базу распределения образуют данные о выпущенной продукции в течении месяца.

Плановая стоимость продукции

Базу распределения образуют данные о выпущенной продукции в течении месяца и ее плановой стоимости. Вид плановых цен устанавливается в разделе НСИ и администрирование - Производство.

Количество работ указанных видов

В качестве базы распределения используются данные об отраженных трудозатратах - выполненных работ в течении месяца. Выполненные работы регистрируются документом Выработка сотрудников.

Норматив расхода данного материала

Базу распределения образуют данные о нормативах потребления распределяемого материала для продукции, выпущенной в текущем месяце. Норматив определяется ресурсными спецификациями, по которым оформлен выпуск продукции.
