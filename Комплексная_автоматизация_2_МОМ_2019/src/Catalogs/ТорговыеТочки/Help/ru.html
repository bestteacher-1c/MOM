Торговые точки

Перечень предназначен для хранения информации о торговых точках, по которым осуществляется оплата торговых сборов, а также для автоматического заполнения уведомления о постановке на учет в качестве плательщика торгового сбора (регламентированная форма ТС-1) и о снятии с учета (Регламентированная форма ТС-2).
Для торговой точки предусмотрена возможность формирования регламентированных форм уведомлений: о постановке на учет в ФНС, об изменении параметров, о снятии с учета.

Ставка торгового сбора может быть рассчитана автоматически. Для этого необходимо:

  • Выбрать тип торговой точки
  • Ввести адрес (либо в формате КЛАДР, либо в произвольном формате, указав код ОКТМО)
  • Указать торговую площадь, если это необходимо
  • Выбрать режим применения налоговой льготы

Расчет торгового сбора для торговой точки производится с квартала, определяемого датой в реквизите Начало использования в обработке Закрытие месяца.
