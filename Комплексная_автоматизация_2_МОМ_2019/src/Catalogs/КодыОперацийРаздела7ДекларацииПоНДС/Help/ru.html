Коды операций раздела 7 декларации по НДС

Справочник предназначен для хранения кодов операций, используемых для заполнения раздела 7 декларации НДС, а также для заполнения реестра документов, подтверждающих льготу по НДС. Новый элемент справочника может быть подобран из классификатора.

Для включения операций в реестр документов, подтверждающих льготу по НДС, необходимо установить флажок Включается в реестр подтверждающих документов

Для заполнения графы 2 "Вид (группа, направление) необлагаемой операции" реестра подтверждающих документов используется значение реквизита Вид необлагаемой операции
