Акт инвентаризации расчетов

Документ предназначен для проведения инвентаризации расчетов с покупателями, поставщиками, банками по кредитам и займам, сотрудниками, бюджетом, внебюджетными фондами другими дебиторами и кредиторами. На основании документа формируются печатные формы ИНВ-17 и ИНВ-22.

На вкладке Основное указывается сведения о проведении инвентаризации: период, причины и реквизиты документа, на основании которого проводится инвентаризация.

На вкладке Счета задается перечень счетов бухгалтерского учета, участвующих в инвентаризации, выводятся итоги задолженности по ним. Выбор счетов для проведения инвентаризации осуществляется по команде Настроить.
С помощью команды Задолженность по счету осуществляется переход к детализации задолженности по счету в разрезе дебиторов/кредиторов.
Для открытия оборотно-сальдовой ведомости по выбранному счету необходимо воспользоваться командой Открытие ОСВ.

На вкладке Задолженность фиксируются результаты инвентаризации. Для этого необходимо указать суммы в колонках Подтверждено, Не подтверждено, Истек срок давности. С помощью команды Открыть ОСВ выполняется открытие оборотно-сальдовой ведомости по счету с отбором по дебитору/кредитору, выбранному в строке.

Члены инвентаризационной комиссии для вывода задаются на вкладке Инвентаризационная комиссия. С помощью команды Подбор доступен выбор членов комиссии из справочника Физические лица. Также возможно указание членов комиссии в произвольной строковой форме.
