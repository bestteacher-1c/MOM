Переоценка ОС

Документ предназначен для отражения переоценки основных средств.

Основное

Основные средства

Основное
  • Организация и Подразделение - организация и подразделение, в которых числятся основные средства;
  • Уценка накопленной амортизации - задается статья, по которой будет отражена уценка накопленной амортизации;
  • Уценка восстановительной стоимости - указывается статья, по которой будет отражена уценка восстановительной стоимости.

Основные средства

На закладке выбираются основные средства, по которым выполняется переоценка, и их стоимость в регламентированном и управленческом учетах.

С помощью команды Подобрать можно добавить несколько основных средств. Для быстрого заполнения списка группой однотипных основных средств необходимо в форме выбора отобрать основные средства по наименованию, выделить несколько строк списка и добавить их в документ с помощью команды Выбрать.
