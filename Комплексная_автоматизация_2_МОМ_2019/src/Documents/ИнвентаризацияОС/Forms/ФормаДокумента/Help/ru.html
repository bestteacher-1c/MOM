Основное


При вводе документа требуется обязательно указать Местонахождение ОС - подразделение организации, в котором проводится инвентаризация основных средств.

В группе Проведение инвентаризации указываются данные, необходимые для заполнения сведений о инвентаризации и причинах ее проведения. В поле Документ вводится наименование документа-основания для проведения инвентаризации. Его можно выбрать из списка или указать вручную.

Основные средства


На закладке Основные средства указывается список основных средств и результаты проведения инвентаризации. Для автоматического заполнения табличной части документа предназначена кнопка Заполнить:

  • Заполнить - По остаткам - заполняется список основных средств по остаткам на дату проведения инвентаризации.
  • Заполнить - Данные учета - обновляются данные колонок Счет учета, Наличие и стоимость по данным учета, исходя из данных информационной базы.
  • Заполнить - Фактические данные - автоматически заполняется колонка Наличие и стоимость по факту по данным учета. В колонке По факту подтверждается соответствие данных учета фактическому наличию основных средств. Фактическая стоимость заполняется автоматически в случае наличия основного средства по данным учета, если данных учета по основному средству нет, Фактическую стоимость можно указать вручную.

При наличии расхождений данных учета с фактическим наличием, для отражения результатов инвентаризации вводятся соответствующие документы на основании. А так же можно воспользоваться ссылкой для создания документа в колонке Отражение результатов инвентаризации. Если инвентаризируемое основное средство было получено в аренду по схеме интеркампани, результаты инвентаризации могут быть отражены только от лица организации владельца объекта ОС.

Список основных средств можно заполнить с помощью сканера штрихкодов или терминала сбора данных.

Инвентаризационная комиссия


На странице Инвентаризационная комиссия перечисляется состав членов инвентаризационной комиссии, председатель выбирается путем установки флага в колонке Председатель.

Печатные формы


Для документа Инвентаризация ОС предусмотрены следующие печатные формы:

  • Инвентаризационная опись ИНВ-1
  • Сличительная ведомость ИНВ-18
  • Приказ о проведении инвентаризации ИНВ-22

Ввод на основании


На основании документа Инвентаризация ОС можно оформить следующие хозяйственные операции:

  • Принятие к учету ОС
  • Перемещение ОС
  • Списание и частичная ликвидация ОС
  • Выбытие арендованных ОС
  • Переоценка ОС
