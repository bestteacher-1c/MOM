Начисление и списание резервов по сомнительным долгам

Документ предназначен для отражения сумм резервов по сомнительным долгам. При проведении документ выполняет начисление или восстановление резервов по сомнительным долгам.

Документ формируется автоматически в результате выполнения этапа закрытия месяца Формирование резервов по сомнительным долгам. После формирования документ возможно изменить вручную.

В документе указывается Организация и Период, по которым выполняется отражение резервов.
В табличной части Дебиторская задолженность отражается дебиторская задолженность организации на конец указанного периода, а также расчет резервов по сомнительным долгам по бухгалтерскому и налоговому учету.
В табличной части Кредиторская задолженность отражается кредиторская задолженность организации на конец указанного периода.

При проведении документа суммы Резерв БУ и Резерв НУ сравниваются с текущими суммами резервов (берутся из данных учета).
Если новая оценка резерва больше текущей оценки, то отражается начисление резерва.
Если новая оценка меньше текущей оценки, то отражается восстановление резерва.

При выполнении команды Заполнить дебиторскую задолженность, табличная часть заполняется дебиторской задолженностью на дату проведения документа, рассчитываются суммы резервов.
Суммы резервов возможно указать вручную. Для этого необходимо установить галочку "Ручная корректировка" и, либо изменить оценку резерва, либо саму сумму резерва.

Лимит расходов НУ

С 2017 года при расчете резервов по сомнительным долгам за отчетные периоды ограничение в НУ равно большей из сумм (п. 4, ст. 266 НК РФ):

  • 10% выручки текущего отчетного периода
  • 10% выручки предыдущего года

При расчете за год ограничение по-прежнему 10% выручки отчетного года (405-ФЗ от 30.11.16).
Лимит расходов НУ заполняется при выполнении команды Заполнить документ или по ссылке Обновить.

Отражать в бухгалтерском учете

Если установлена галочка, то рассчитанная сумма резерва БУ будет учтена при начислении/восстановлении резервов по сомнительным долгам по бухгалтерскому учету.

Отражать в налоговом учете

Если установлена галочка, то рассчитанная сумма резерва НУ будет учтена при начислении/восстановлении резервов по сомнительным долгам по налоговому учету.
Если используется ПБУ 18/02, то независимо от галочки, разница бежду БУ и НУ отразится на постоянной или временной разнице, в соответствии с настройками учетной политики организации.

Заполнение табличной части Дебиторская задолженность

Табличная часть Дебиторская задолженность заполняется текущим состоянием дебиторской задолженности организации на основании данных оперативных регистров.

Вся задолженность делится по видам задолженности:

  • Задолженность контрагентов в рублях
  • Задолженность контрагентов в валюте и у.е.
  • Выданные авансы
  • Займы выданные

Так же можно добавить дебиторскую задолженность вручную. В этом случае её вид задолженности будет установлен в Прочее.

Для каждой строки дебиторской задолженности заполняется:

  • Аналитика учета - Контрагент, договор контрагента, расчетный документ. Или в случае с займами выданными - Тип суммы (Основной долг или Проценты).
  • Задолженность - в рублях и в валюте взаиморасчетов.
  • Срок оплаты - дата возникновения задолженности и количество дней.

Сумму обеспечения можно заполнить вручную. При этом, если для одной задолженности (расчетного документа) существует несколько задолженностей с разными строками, то указанная сумма обеспечения распределится по это задолженности автоматически.
Сумма встречной задолженности распределяется автоматически по задолженности контрагента в целом на основании данных табличной части Кредиторская задолженность.

Колонка Оценка заполняется автоматически в зависимости от Вида задолженности в соответствии с учетной политикой организации.

Можно изменить суммы резерва вручную. Для этого нужно установить галочку Ручная корректировка и изменить Оценку (процент) или саму сумму резерва.
К изменению доступны суммы Резерв БУ (в валюте регламентированного учета) и Резерв (в валюте взаиморасчетов).
Сумма Резерв НУ недоступна для изменения и всегда считается по правилам налогового учета.

Заполнение табличной части Кредиторская задолженность

Табличная часть Кредиторская задолженность заполняется текущим состоянием кредиторской задолженности организации на основании данных оперативных регистров.
Можно вручную изменить сумму кредиторской задолженности, при этом измененная строка отметится галочкой Ручная корректировка.
При снятии галочки Ручная корректировка сумма данной строки изменится на данные учета.
Можно вручную добавить новую строку кредиторской задолженности.
