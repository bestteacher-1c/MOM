Начисление и списание резервов предстоящих расходов

Документ предназначен для отражения сумм резервов (оценочных обязательств) по объектам учета. При проведении документ выполняет начисление или списание резервов по объектам.

В документе указывается Организация и Вид резервов, по которым выполняется отражение резервов.
В табличной части Резервы выбираются Объекты учета резервов. В поле Новая сумма указывается актуальная оценка резерва (оценочного обязательства) по объекту.

При проведении документа Новая сумма сравнивается с Текущей суммой (заполняется по данным ИБ).
Если новая оценка резерва больше текущей оценки, то отражается начисление резерва.
Если новая оценка меньше текущей оценки, то отражается списание резерва.
Также сумма резерва может быть оставлена без изменений.

При выполни команды Заполнить, в табличную часть добавляются Объекты учета резервов, актуальные на дату проведения документа, заполняется текущая оценка резервов. Новую оценку резервов необходимо указать вручную.

Также новая оценка резервов может быть загружена из табличного документа. Для этого необходимо воспользоваться командой Загрузить из файла…
