Основное
  • Операция - возможны следующие виды операций: Списание основных средств, Списание на основании инвентаризации;
  • Организация и Подразделение - организация и подразделение, в которых числятся основные средства;
  • Частичная ликвидация - позволяет выполнить частичное списание основных средств;
  • Статья и аналитика расходов - статья расходов или статья активов и соответствующая ей аналитика, по которой отражаются расходы от списания основных средств;
  • Причина списания - причина списания основных средств (будет указана в печатных формах).

Основные средства

На закладке указываются списываемые основные средства.
При частичном списании по каждому основному средству требуется указать ликвидируемую величину из восстановительной стоимости.

С помощью команды Подобрать можно добавить несколько основных средств. Для быстрого заполнения списка группой однотипных основных средств необходимо в форме выбора отобрать основные средства по наименованию, выделить несколько строк списка и добавить их в документ с помощью команды Выбрать.

Приходуемые материальные ценности

На закладке указываются материальные ценности (например, драгоценные металлы), возникающие при списании основных средств. Поступление материальных ценностей осуществляется из группы команд Создать на основании при помощи документа Прочее оприходование товаров.

Печатные формы

Для документа доступны печатные формы:

  • Акт о списании объекта основных средств ОС-4 (кроме автотранспортных средств).
  • Акт о списании автотранспортных средств ОС-4а (формируется при списании основных средств с признаком Автотранспорт).
  • Акт о списании групп объектов основных средств ОС-4б (формируется при списании более, чем одного основного средства).
