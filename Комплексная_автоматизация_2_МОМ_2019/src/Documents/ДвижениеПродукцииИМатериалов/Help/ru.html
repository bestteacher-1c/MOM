Движение продукции и материалов

Документ предназначен для отражения в системе движения продукции, материалов и полуфабрикатов между складами и цеховыми кладовыми в рамках выполнения следующих операций:

  • Передача продукции из кладовой
  • Передача материалов в кладовую
  • Перемещение полуфабрикатов
  • Перемещение полуфабрикатов между филиалами
  • Возврат материалов из кладовой

Документ может создаваться как самостоятельно, так и по распоряжениям. В качестве распоряжений могут выступать заказы материалов в производство.
Основное рабочее место для работы с документом расположено в меню Производство - Материальный учет - Движение продукции и материалов.

Документ доступен, если включена опция Производство - Управление производством "Версия 2.2".

Передача продукции из кладовой

Используется для отражения передачи произведенной продукции из цеховой кладовой на склад. Передача осуществляется без распоряжения.

Передача материалов в кладовую

Операция используется для описания передачи материала со склада в цеховую кладовую. Передача осуществляется без распоряжения или по заказу материалов в производство. При передаче по распоряжению возможно использование строк сверх заказа.

Перемещение полуфабрикатов

Предназначено для отражения перемещения между цеховыми кладовыми разных подразделений. Выполняется без распоряжений.

Перемещение полуфабрикатов между филиалами

Предназначено для отражения перемещения между цеховыми кладовыми и филиалами организации. Выполняется без распоряжений. Операция доступна, если включена функциональная опция НСИ и администрирование - Предприятие - Обособленные подразделения на отдельном балансе.

Возврат материалов из кладовой

Используется для возврата на склад из цеховой кладовой материалов, ранее переданных туда. Возврат осуществляется без распоряжения.

Поступление товаров на ордерном складе


Оформление приходного ордера по накладной доступно в тех случаях, когда:

  • Накладная оформлена не по заказу;
  • При значении функциональной опции Оформление приходного ордера доступно "После оформления накладной" (в разделе НСИ и администрирование - Склад и доставка - Склад).


При использовании настройки оформления приходного ордера После оформления накладной, состав и количество товара в приходном ордере будет заполнено по накладной, независимо от того, по заказу накладная или нет.
