Рабочее место

В Рабочем месте отображется список невыполненных распоряжений, по которым необходимо оформить документы или не завершен процесс отгрузки (приемки) по ордерной схеме.

Для рабочего места доступны два режима формирования: По распоряжениям и По номенклатуре.

В режиме формирования По номенклатуре позиции распоряжений, участвующие в заполнении документов, ограничиваются номенклатурой (характеристикой) выбранных строк.

По команде Оформить по распоряжениям создается документ и заполняется позициями распоряжений, по которым не оформлены документы. По команде Оформить по отгрузке (приемке) создается документ движения и заполняется отгруженными (принятыми) по ордерам позициями.

Для индикации состояния оформления документов ордеров, оформленных по распоряжениям, предусмотрены две информационные колонки с пиктограммами. Пояснение назначения пиктограмм приведено в соответствующих колонкам отборах.
