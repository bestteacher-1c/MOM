Форма выбора распоряжения

В форме отображается список товаров из заказов и расходных складских ордеров, по которым не оформлены документы передачи.
Переключатель Все отгруженные/Все заказанные позволяет автоматически заполнять количество товаров, которые необходимо перенести в документ либо заказанными, либо отгруженными товарами.
Флажками отмечаются те строки, которые необходимо перенести в документ. По команде Перенести в документ информация по отмеченным строкам будет обновлена в документе.
