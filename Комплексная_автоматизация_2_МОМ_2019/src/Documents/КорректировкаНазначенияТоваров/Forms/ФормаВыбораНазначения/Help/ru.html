Выбор назначения

Форма предназначена для выбора заказа. В форме отображаются заказы, в которых зафиксирована обособленная потребность: указано действие Обеспечивать обособленно или Отгрузить обособленно.

При использовании направлений деятельности может быть зафиксирована обособленная потребность:

  • по конкретному заказу направления деятельности (вариант обособления: по заказам направления),
  • по направлению деятельности, без детализации до заказа (вариант обособления: по направлению деятельности в целом).


В этом случае в форме можно выбрать следующие виды назначений:

  • заказ без указания направления деятельности,
  • заказ с указанием направления деятельности,
  • направление деятельности без детализации до заказов.

Один заказ может встречаться в списке несколько раз с различными направлениями деятельности, если в данном заказе направление деятельности было изменено.

При использовании нескольких организаций флаг Только заказы организации позволяет отобрать в списке заказы, оформленные от лица конкретной организации. В этом случае назначения с вариантом обособления по направлению деятельности в целом будут скрыты.
