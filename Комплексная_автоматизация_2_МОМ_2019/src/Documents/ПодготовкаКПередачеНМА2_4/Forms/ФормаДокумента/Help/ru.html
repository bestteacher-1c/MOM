Подготовка к передаче НМА

Документ предназначен для отражения подготовки к переходу права собственности для объекта нематериальных активов или расходов на НИОКР.

Для оформления фактической передачи необходимо создать документ Реализация услуг и прочих активов с операцией Реализация внеоборотных активов. Документ можно создать, используя механизм ввода на основании.

Основное

Нематериальные активы

Основное
  • Организация - организация, в которой принят к учету объект нематериального актива или расхода на НИОКР;
  • Подразделение - подразделение организации, ответственное за продаваемый актив;
  • Партнер, Контрагент - покупатель объекта нематериального актива.

Нематериальные активы

На закладке указываются нематериальные активы и расходы на НИОКР, которые планируется передать.

С помощью команды Подобрать можно добавить несколько нематериальных активов. Для быстрого заполнения списка группой однотипных нематериальных активов необходимо в форме выбора отобрать нематериальные активы по наименованию, выделить несколько строк списка и добавить их в документ с помощью команды Выбрать.
