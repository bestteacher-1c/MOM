Регистрация порядка налогообложения по налогу на имущество

Документ предназначен для указания особого порядка налогообложения по налогу на имущество для перечня объектов эксплуатации и основных средств.

Основное


На закладке Основное уточняются Организация, для объектов налогообложения которой осуществляется указание порядка налогообложения по налогу на имущество.

Значения возможные к выбору в полях Порядок налогообложения и Налоговая база зависят от указанного вида имущества и обязательны к заполнению.
Выбор вида имущества доступен только для документов, созданных до 2019 года. После 2019 года документ используется только для недвижимого имущества.

  • Особые льготы не установлены - используется для недвижимого имущества или движимого имущества принятого к учету до 2013 года;
  • Подлежит налогообложению, применяется ставка для движимого имущества - используется для движимого имущества, принятого с 2013 года начиная с 2018 года (не используется с 2019 года);
  • Подлежит налогообложению, применяется общая ставка - используется для движимого имущества, принятого с 2013 года начиная с 2018 года, принятого на учет в результате: реорганизации или ликвидации юридических лиц, передачи, включая приобретение, имущества между лицами, признаваемыми взаимозависимыми;
  • Освобождается от налогообложения - используется для любого вида имущества;
  • Облагается по сниженной ставке - используется для любого вида имущества;
  • Не подлежит налогообложению - используется для недвижимого имущества или движимого имущества принятого к учету до 2013 года.

Налоговый орган


Реквизиты закладки Налоговый орган используются при необходимости уточнения постановки объекта на учет в налоговом органе, отличающемся от указанного для организации по умолчанию.

Недвижимое имущество


На закладке указывается недвижимое имущество, для которого устанавливается особый порядок налогообложения.
Дополнительные параметры:

  • Доля в праве собственности, если имущество находится в общей (долевой или совместной) собственности
  • Доля площади, если кадастровая стоимость помещения определена исходя из доли площади помещения в общей площади здания
  • Доля стоимости, относящаяся к территории субъекта РФ.

Отражение расходов


На закладке Отражения расходов в строках табличной части необходимо указать Подразделение и Статью и аналитику расходов, на которые будет распределяться налог на имущество.
Коэффициент пропорции (может быть, как дробным, так и целым числом) определяет, какая часть будет отнесена на данные Подразделение и Статью/аналитику расходов указанные в строке.
Если способы отражения расходов не указаны в документе, то при расчете налога будут действовать ранее установленные способы. Если ранее тоже не были заданы способы, то при расчете будет выдано сообщение о необходимости ввода способа отражения расхода.
