Списание из эксплуатации

Документ предназначен для списания находящихся в эксплуатации товарно-материальных ценностей.

В табличной части обязательна к заполнению колонка Партия. Для каждой из строк необходимо указать ссылку на партию ТМЦ в эксплуатации, которая автоматически формируется при оформлении документа передачи в эксплуатацию.
При последовательном заполнении строки документа колонка Партия автоматически заполняется по данным учета. При необходимости выбора другой партии, значение можно отредактировать, при этом список выбора ограничивается доступными, соответствующими документу и текущей строке, партиями.

Бухгалтерские проводки формируются отложено при закрытии месяца документом Погашение стоимости ТМЦ в эксплуатации.

Печатные формы

Печатные формы

Для документа Списание из экплуатации можно вывести на печать следующие печатные формы:

  • Акт на списание (МБ-8) – акт на списание малоценных и быстроизнашивающихся предметов;

Стоимостная оценка и амортизация списываемых товарно-материальных ценностей в форме Акт на списание (МБ-8) определяется при закрытии месяца.

Примечание. Для корректного отображения сумм в графе амортизации печатной формы МБ-8 для списываемой номенклатуры с разными характеристикми
рекомендуется для каждого вида характеристик вводить отдельные документы передачи.
