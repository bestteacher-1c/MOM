Партии производства

Форма предназначена для выбора партий производства, на которые можно отнести затраты. Включение флага Показывать только те партии, работы по которым выполнялись в текущем месяце означает, что будут показаны только те партии, выполнение работ по которым пришлось на данный месяц.

Вариант подбора

Вариант подбора
  • По продукции - форма адаптируется для упрощения поиска партии производства, путем анализа данных о выпущенной продукции;
  • По этапам - форма адаптируется для упрощения поиска партии производства, путем анализа данных о выполненных этапах производства.
