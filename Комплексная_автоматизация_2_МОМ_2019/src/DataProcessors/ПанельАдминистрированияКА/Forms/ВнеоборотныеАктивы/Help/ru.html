Внеоборотные активы

Используя данный раздел панели администрирования можно настроить параметры ведения учета внеоборотных активов.

Выбор версии учета внеоборотных активов версии - для учета внеоборотных активов необходимо выбрать требуемую версию: версия 2.2 или версия 2.4. Если используется обе версии то нужно указать с какой даты начинает действовать версия 2.4.

Отображать команды для ведения учета внеоборотных активов версии 2.2 При отключеной опции возможность создания документов версии 2.2 будет недоступна. При этом ранее созданные документы будут отображаться в журналах документов ОС и НМА.
Опция доступна, если ранее велся учет внеоборотных активов версии 2.2.

Объекты строительства - Позволяет использовать объекты строительства для накопления затрат.

Лизинг - использовать функциональность получения предметов лизинга, начислений и оплаты лизинговых услуг.

Учет платежей в системе "Платон" - позволяет оформлять документы Отчет оператора системы "Платон" и вести учет платежей при расчете транспортного налога.

Данная версия используется для совместимости. Для новых баз или для баз, в которых ранее не велся учет внеоборотных активов, рекомендуется сразу использовать учет внеоборотных активов версии 2.4.

Учет внеоборотных активов в международном учете - позволяет использовать отдельные документы для учета внеоборотных активов в международном учете.

Отражение начисленной амортизации регламентированного учета в управленческом учете - позволяет погашать накопленные затраты внеоборотных активов в управленческом учете на основании начисленной амортизации в регламентированном учете.

Учет внеоборотных активов в управленческом учете

  • По стандартам международного учета - амортизация начисляется с даты документа, изменение параметров влияет на начисление амортизации с даты документа;
  • По стандартам регламентированного учета - амортизация начисляется со следующего месяца, изменение параметров влияет на начисление амортизации со следующего месяца.
