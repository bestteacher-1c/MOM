Границы включения сделок в уведомление

Форма позволяет редактировать данные регистра сведений "Границы контролируемости сделок" за год, по которому предоставляется уведомление.

Группа общего ограничения

Сделки будут отнесены в разряд контролируемых, если сумма сделок за отчетный год будет превышать сумму указанную в данном разделе. Это является одним из необходимым условием для включения сделок в уведомление (но не достаточным).

Соответствует особенности отнесения в регистре Минимальная сумма для включения в уведомление.

Группа для контроля сделок с иностранными независимыми лицами

Сделки, совершаемые иностранным контрагентом, будут отнесены в разряд контролируемых, если сумма сделок за отчетный год будет превышать сумму указанную в данном разделе при выполнении одного из следующих условий:

  • Иностранный контрагент зарегестрирован в оффшоре;
  • Предметом сделки является товар мировой биржевой торговли.

Соответствует особенности отнесения в регистре Независимые иностранные лица.

Группа ограничения для контроля сделок с контрагентами РФ

Сделки, совершаемые российским взаимозависимым контрагентом, будут отнесены в разряд контролируемых при выполнении одного из следующих условий:

  • Одно из взаимозависимых лиц является резидентом особой экономической зоны и сумма сделок за отчетный год превышает сумму указанную для данного показателя (по признаку особой экономической зоны);
  • Одно из взаимозависимых лиц является плательщиком налога на прибыль и сумма сделок за отчетный год превышает сумму указанную для данного показателя (по принзнаку уплаты налога на прибыль);
  • Одно из взаимозависимых лиц является плательщиком ЕНВД или ЕСХН и сумма сделок за отчетный год превышает сумму указанную для данного показателя (по признаку применения спецрежимов);
  • Одно из взаимозависимых лиц является плательщиком НДПИ, сделка совершается по товарам облагаемым НДПИ и сумма сделок за отчетный год превышает сумму указанную для данного показателя (по признаку уплаты НДПИ);
  • Сумма сделок между взаимозависимыми лицами за отчетный год превышает сумму указанную в качестве общего ограничения для контрагентов РФ (Общее ограничение).

При отнесении сделки к контролируемым по условию применения спецрежима ЕНВД, следует обращать внимание, что не только одно из взаимозависимых лиц должно облагаться ЕНВД, но и в самом документе должна быть отметка, что данная сделка совершена с применением ЕНВД.
