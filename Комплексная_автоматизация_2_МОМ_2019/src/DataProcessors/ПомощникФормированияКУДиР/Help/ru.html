Помощник формирования книги учета доходов и расходов

Помощник предназначен для упрощения формирования Книги учета доходов и расходов.
В зависимости от настроенной учетной политики организации (использующей или не использующей упрощенную схему налогообложения) вид формы различается для объекта налогообложения Доходы и Доходы минус расходы.

Подготовка данных по доходам и расходам
  • настройка статей расходов,
  • разделение номенклатуры на материалы и товары для раздельного учета расходов,
  • выполнение процедуры признания расходов,
  • выполнение процедуры сторнирования доходов,
  • выполнение процедуры расчета налога при УСН.

Проверка и анализ книги учета доходов и расходов
  • формирование отчетов для анализа расходов (Расходы по товарам/материалам/ОС и НМА, а также Прочим расходам),
  • проведение ручных операций по корректировке Книги учета доходов и расходов.

Анализ полученного результата
  • Книга учета доходов и расходов,
  • Справка расчет налога УСН,
  • Декларация по УСН.

Настройка статей расходов


Непосредственная настройка статей расходов осуществляется в отдельном списке. Для каждой статьи расходов требуется установить необходимое значение Признавать/Не признавать расходом при УСН. Причем установленные настройки будут действовать, только для текущих операций. Для корректировки ранее признанных расходов необходимо повторно выполнить признание.

Если в статье расходов определено, что они относятся к деятельности облагаемой ЕНВД, то такие расходы в УСН не признаются. Статьи расходов с признаком ОСНО и ЕНВД признаются в УСН в соответствии с долей доходов, относящейся к УСН от общей суммы доходов. Расчет доли выполняется ежемесячно.

Порядок признания расходов для товаров и материалов


Предусмотрены различные сценарии порядка признания расхода для товаров и материалов. Если товары признаются расходами по факту их продажи конечному потребителю, то для признания расходами материалов достаточно их поступления на склад и последующей оплаты. Выделение материалов осуществляется на основании счета учета номенклатуры. Если он относится к группе счетов Материалы (10), то данная номенклатура будет признана материалом и расход по ней будет осуществлен при соблюдении двух условий (поступление расхода и оплата).

Если в последующем материал списывается на статью расходов, в которой указано, что расходы относятся к деятельности облагаемой ЕНВД, то происходит сторнирование расходов. При списании расходов на статью расходов с признаком ОСНО и ЕНВД, сторнирование расходов выполняется частично, в соответствии с долей относящейся к деятельности, облагаемой ЕНДВ.

При выполнении процедуры признания расходов они автоматически заносятся в Книгу учета доходов и расходов. Если в текущем периоде признание расходов на всю сумму невозможно, то оставшаяся сумма доступна для признания в будущих периодах.
Признание расходов предусмотрено помесячно и за квартал.
Расчет налога производится поквартально (для налога по УСН рассчитывается авансовый платеж) и за весь год.

Проверка отражения прочих доходов и расходов


После выполнения операций сторнирования доходов и признания расходов, доступен быстрый анализ записей в книге учета доходов и расходов. Позволяет увидеть по каким статьям (доходов, расходов, активов и пассивов) формировались данные в книге, общий оборот по статье и какая часть суммы будет отражена в итоговых данных. При выборе одной из статей, в правой части формы будут отражены более подробные данные по прошедшим операциям.
Рабочее место имеет 3 раздела:

  • Прочие расходы - признаются для целей УСН в зависимости от свойства статьи расходов "Признавать расходами при УСН";
  • Прочие доходы - прочие поступления денежных средств с указанной статьей доходов (по умолчанию признаются для целей УСН);
  • Прочие поступления денежных средств - прочие поступления денежных средств с указанной статьей активов/пассивов (по умолчанию не признаются доходами для целей УСН).
